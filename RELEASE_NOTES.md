# Desktop Task View v0.1.0

First public release.

## Features

- Single-click an empty desktop area to minimize regular windows.
- Single-click the desktop again to restore the windows minimized by this app.
- Double-click an empty desktop area to open Windows Task View.
- Tray menu with enable/disable, manual Task View, manual desktop toggle, and exit.
- Backup shortcuts:
  - `Ctrl + Alt + F11`: Windows desktop toggle.
  - `Ctrl + Alt + F12`: Task View.

## Notes

This is a small personal Windows 11 utility made by a beginner through vibe coding with Codex + GPT.

It exists because I wanted a tiny workflow helper for my own desktop, learned by building, and polished it step by step until it became useful.
